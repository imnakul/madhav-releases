## What it does

- **Sidebar panel** — slides out from the right edge of your screen. Hover to open, moves away when you're done.
- **Global shortcuts** — `Alt+S` captures selected text from any app. `Alt+Shift+S` OCRs any part of your screen.
- **Telegram sync** — send a message to your bot, it shows up in the sidebar instantly. Works both ways.
- **Notes, Tasks, Ideas, Bookmarks** — with markdown, reminders, pinning, archiving, and search.
- **Auto-updates** — new versions download in the background and install when you quit.
- **No account needed** — works fully offline. Telegram link is optional.

## Getting started

1. Download and install using the link above
2. Madhav appears in your system tray — hover the right edge of your screen to open the panel
3. Start adding thoughts

**Optional — Telegram sync (1 step):**
Open **Settings → Telegram**, scan the QR code or click the link. Done — notes sync instantly between your desktop and your Telegram bot.

## Features

<details>
<summary><strong>Capture</strong></summary>

- Hover panel — slides out from screen edge, disappears when you're done
- Hotkey mode — toggle with a custom shortcut
- `Alt+S` — captures selected text from any app on your desktop
- `Alt+Shift+S` — opens a snip overlay, OCRs the region you draw
- `Alt+C` — floating quick-capture window usable from any app
- Smart category detection — URLs → Bookmark, bullet lists → Task, everything else → Uncategorized

</details>

<details>
<summary><strong>Notes & Tasks</strong></summary>

- 5 categories: Note · Task · Idea · Bookmark · Uncategorized
- Full Markdown with live preview
- Pin up to 5 notes to the top
- Archive without deleting
- Rich link previews for bookmarks (favicon, title, og:image)
- Reminders with OS notification + Telegram alert
- Drag to reorder
- Search with highlighted results
- Completed tasks collapse into a separate section

</details>

<details>
<summary><strong>Telegram Bot</strong></summary>

- Send any message → appears as a note on your desktop
- Create on desktop → appears in Telegram
- Manage with inline buttons (complete, edit, delete, remind)
- Commands: `/tasks` `/notes` `/ideas` `/all` `/remind`
- No BotFather setup needed — just link your account in Settings

</details>

<details>
<summary><strong>Customization</strong></summary>

- Background image or video in the panel
- Adjustable opacity and blur
- Custom keyboard shortcuts for everything
- Sidebar width, panel behavior (hover / fixed / hotkey)
- Sound effects toggle

</details>

## Auto-updates

Madhav checks for updates on startup. When one is available it downloads in the background and installs when you quit — no action needed. You'll see a notification banner inside the app.
